# Turnaround Time Generator

## Getting Started

1. Ensure you have `python` installed
2. Install necessary dependencies by navigating to this directory and running the following commands:<br>
   `python3 -m venv .venv`<br>
   `source .venv/bin/activate` on Linux or <br>
   `source .venv/Source/activate` on Windows <br>
   `pip install -r requirements.txt` <br>

For an example, run the following command:<br>
`python3 turnaround.py ./auditlogs/auditlog.json`
